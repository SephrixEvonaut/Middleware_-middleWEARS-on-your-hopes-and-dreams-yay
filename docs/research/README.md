# Research Findings

This folder contains AI research responses from Claude and ChatGPT.

## File Naming Convention

- `phase0-{topic}.md` - Phase 0 (Architecture) research
- `phase1-{topic}.md` - Phase 1 (Raw Input) research
- `phase2-{topic}.md` - Phase 2 (Device Parsers) research
- `phase3-{topic}.md` - Phase 3 (Anti-Cheat) research
- `phase4-{topic}.md` - Phase 4 (Cross-Platform) research

## How to Use This Folder

1. **Run AI prompts** from MIDDLEWARE_ROADMAP.md
2. **Save responses** here as markdown files
3. **Extract decisions** → Update ../DECISIONS.md
4. **Extract code** → Implement in middleware/src/
5. **Update status** → Mark tasks complete in roadmap
